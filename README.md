# pandanet-status

pandaNet status is an ongoing personal project to build a dashboard to view the status of my self-hosted infrastructure.
[[Documentation/pandanet-wireframe.png]]

The dashboard is being built in QT Quick, with the intention of both desktop and web support via WebAssembly. It is also planned to integrate with the Tailscale API.
